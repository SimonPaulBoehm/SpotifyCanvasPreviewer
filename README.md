# Spotify Canvas Previewer

Preview how your **Spotify Canvas videos** will look directly in the Spotify mobile layout.

This tool helps artists and producers quickly check if their **Canvas (9:16 loop video)** fits correctly before uploading to Spotify for Artists.

---

## Features

- Mobile-style Spotify preview
- Upload your Canvas video
- Instant playback preview
- Simple and lightweight

## How to Use

1. Open the previewer  
2. Upload your Canvas video  
3. Check how it appears in the Spotify layout  

---

## Canvas Requirements (Spotify)

- **Aspect Ratio:** 9:16  
- **Length:** 3–8 seconds  
- **Format:** MP4  
- **Resolution:** 720×1280 recommended
